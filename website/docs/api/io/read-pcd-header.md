# readPCDHeader

```ts
readPCDHeader(data)
```

Read the header of any PCD file data.

**Parameters:**

| Name | Type          | Default | Description    |
| ---- | ------------- | ------- | -------------- |
| data | `ArrayBuffer` |         | The file data. |

**Returns:**

| Name   | Type                                                  |
| ------ | ----------------------------------------------------- |
| header | [`PCDHeader`](/docs/api/io/read-pcd-header#pcdheader) |
